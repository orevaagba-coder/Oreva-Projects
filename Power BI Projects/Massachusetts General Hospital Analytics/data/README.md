# Data

Raw CSV files used in the MGH Hospital Analytics dashboard. This is 
synthetic healthcare data generated by Synthea, sourced via Maven Analytics.

## Files

- **patients.csv** — Patient demographics (age, gender, race, ethnicity, location)
- **encounters.csv** — Hospital visits (dates, encounter class, costs, payer)
- **procedures.csv** — Procedures performed during encounters (codes, costs)
- **payers.csv** — Insurance providers
- **organizations.csv** — Hospital/organization details

## Notes

- Data spans 2011–2022
- All data is synthetic and does not represent real patients
- See `data_dictionary.csv` for field definitions (not included in dashboard model)
