# Massachusetts General Hospital Analytics Dashboard

Power BI dashboard analyzing 11 years (2011–2022) of synthetic hospital 
data from Massachusetts General Hospital, covering admissions, patient 
demographics, and financial performance.

## Data Source
Synthetic healthcare data generated by Synthea, sourced via Maven Analytics.
- patients.csv, encounters.csv, procedures.csv, payers.csv, organizations.csv

## Pages
1. **Home** — Cover page with navigation
2. **Admissions Overview** — Encounter trends, weekday/time-of-day patterns
3. **Patient Demographics** — Age, gender, race, geographic distribution
4. **Financial Analysis** — Revenue, payer mix, cost drivers
5. **Key Insights** — Summary findings and recommendations

## Key Findings
- 27,891 encounters across 974 patients (avg 28.6 per patient)
- 64.68% of patients are seniors (65+)
- $101.51M total revenue; only 30.6% covered by insurers
- Electrical cardioversion is the highest-cost procedure ($36M)
